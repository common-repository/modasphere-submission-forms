Modasphere Submission Form is official plugin developed by Casting Networks LLC for Modasphere's clients, who would like to use talent submission form on their web sites. This plugin is bridge between Modasphere and Wordpress systems. Its main feature is creating web site submission forms and sending filled out information to Modasphere CRM.

== Installation ==

1. Download Modasphere Submission Form plugin on the PC
2. Upload the `modasphere-submission-forms` folder to the `/wp-content/plugins` directory
3. Activate the plugin through the 'Plugins' menu in WordPress. "Submission Forms" menu item is created in the admin menu.
4. Click on the "Submission Forms" menu item.
5. Fill in all fields of "AUTH" section, click on "Save" button.
6. Go to "SUBMISSION FORMS" section and fill out all fields. Click on "Save" button.
7. In "SUBMISSION FORMS SETTINGS" section user can change 'description' field, add new fields with help text to submission form and tick form for display.
8. Click on the 'house' icon in the left corner and check all applied changes on the website
